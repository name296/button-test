버튼 컴포넌트 시스템

다양한 상태와 팔레트를 지원하는 접근성 중심의 버튼 컴포넌트입니다.

Primary1 팔레트

테마의 기본 브랜드 색상을 사용하는 버튼들입니다.

Primary2 팔레트

테마의 변형 색상을 사용하는 버튼들입니다. Gray 색상과 조합하여 다양한 대비를 제공합니다.

Secondary1 팔레트 (Gray 기반)

테마에 독립적인 Gray 색상을 기반으로 하는 보조 버튼들입니다. 모든 테마에서 일관된 색상을 유지합니다.

Secondary2 팔레트 (Gray 변형)

Gray 색상의 변형을 사용하는 보조 버튼들입니다. 다양한 회색조로 계층적 인터페이스를 구현합니다.

Primary3 팔레트 (이중 배경 + 그라데이션)

이중 배경 시스템과 그라데이션 효과를 사용하는 고급 버튼들입니다. z-index가 다른 두 배경 레이어로 구성됩니다.

Secondary3 팔레트 (이중 배경 + 아이콘)

이중 배경 시스템으로 아이콘 영역에 별도 배경을 제공하는 버튼들입니다. 아이콘과 텍스트가 다른 배경을 가집니다.
